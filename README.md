# gxeditor

#### 项目介绍
gxeditor(Game Xml Editor)，游戏Xml编辑器。旨在方便的编辑、校验游戏xml文件。
也可以生成cpp读取模版的代码。

#### 使用文档

https://uhziel.gitbook.io/gxeditor

#### 软件下载地址

https://github.com/uhziel/gxeditor/releases 下载有"Lastest release"标记的  gxeditor-setup-xxx.exe

下载安装好，以后有稳定版本更新，会自动升级。

#### 开发环境安装和测试

```bash
npm install
npm start
```

#### 打包和发布

```bash
npm run pack
npm run dist
```
